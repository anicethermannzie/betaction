## Methodology

### Framework: Agile/Scrum
- **Sprint Length:** 2 weeks
- **Ceremonies:** Sprint Planning (Monday), Daily Standups, Sprint Review (Friday), Retrospective
- **Tools:** GitHub Projects (Scrum board), GitHub Issues (backlog), Conventional Commits
- **Branching:** main (production), develop (staging), feature/* (development)
- **Documentation:** docs/sprints/ (sprint records), docs/standups/ (daily logs)
